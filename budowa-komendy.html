Budowa komendy

Zaczynamy od zaimplementowania czegoś, np. `EmbedBuilder`: 

const { EmbedBuilder } = require('discord.js');
// Przecinek dajemy tylko jeśli jest więcej niż jedna rzecz, a na końcu, jeśli użyto {}, musi być średnik.

Następnie kod komendy musi zaczynać się od `module.exports`:

module.exports = {
    name: 'nazwa_komendy', // Najlepiej nazwa taka jak pliku i musi być przecinek na końcu
    description: 'opis komendy co ona robi i takie tam',
    usage: 'użycie komendy np. !ban @nick',
    // Teraz co użycie komendy zrobi
    execute: async(message, args) => {
        // Tutaj już co ma się zadziać, np.
        message.channel.send('wiadomość, którą ma wysłać');
        // A jeśli ma odpowiedzieć graczowi, to coś takiego:
        message.reply('odpowiedź');
    }
};

I tak naprawdę powinna wyglądać standardowa komenda, która coś pisze. A jeśli ma coś innego robić, to zapraszam do projektowanie komend 2.
