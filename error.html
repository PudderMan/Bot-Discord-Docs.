Errory które mozesz spotkac przy budowaniu bota

Błędy API (REST)

Te błędy otrzymasz bezpośrednio od serwera Discorda. Mają one numery, które pomagają w diagnozie.

Błędy ogólne (40xxx)

  • 10008: Unknown Member - Próbujesz odwołać się do użytkownika, który nie istnieje na danym serwerze.
  • 40001: Unauthorized - Używasz nieprawidłowego tokenu bota, który nie przeszedł autoryzacji.
  • 40002: Account Disabled - Konto bota zostało wyłączone.
  • 40003: Invalid Token - Token bota jest niepoprawny lub nieaktualny.
  • 40004: Rate Limited - Bot wysłał zbyt wiele zapytań w krótkim czasie.
  • 40005: Request Time Out - Zapytanie do API trwało zbyt długo.
  • 40006: Invalid Guild - Próbujesz wykonać akcję na serwerze, który nie istnieje lub bot nie ma do niego dostępu.
  • 40007: Invalid Channel - Próbujesz wykonać akcję na kanale, który nie istnieje lub jest niedostępny.

Błędy uprawnień (50xxx)

  • 50001: Missing Access - Brak uprawnień dostępu do danego zasobu, np. próba wysłania wiadomości na kanale, do którego bot nie ma dostępu.
  • 50002: Cannot Execute - Nie można wykonać danej akcji. Np. próba edytowania wiadomości innego użytkownika lub usunięcia zbyt starej wiadomości.
  • 50003: Missing Permissions - Bot nie ma uprawnień do wykonania danej akcji, np. wyrzucenia użytkownika, który ma wyższą rolę.
  • 50005: Cannot Send Messages - Bot nie ma uprawnień do wysyłania wiadomości na danym kanale.
  • 50006: Cannot Send Empty Message - Próba wysłania pustej wiadomości (bez treści, załączników, osadzonych elementów).
  • 50007: Cannot Kick/Ban User - Nie można wyrzucić/zbanować użytkownika, ponieważ ma wyższą rolę.
  • 50008: Cannot Edit Role - Próba edytowania roli, która jest wyższa od roli bota.
  • 50013: Cannot Edit Permission - Brak uprawnień do zmiany uprawnień.
  • 50028: Invalid HTTP Header - Niepoprawny nagłówek w zapytaniu HTTP.

Błędy zasobów (30xxx)

  • 30001: Maximum Servers - Bot osiągnął maksymalną liczbę serwerów, na których może być (maksymalnie 100 bez włączonego sharding'u).
  • 30005: Maximum Guild Channels - Serwer osiągnął maksymalną liczbę kanałów (500).
  • 30007: Maximum Guild Roles - Serwer osiągnął maksymalną liczbę ról (250).
  • 30008: Maximum Guild Emojis - Serwer osiągnął maksymalną liczbę emoji (50/250).
  • 30010: Maximum Webhooks - Serwer osiągnął maksymalną liczbę webhooków (50).

Błędy Gateway (WebSocket)

Te błędy to kody zamknięcia połączenia, które dostajesz, gdy połączenie WebSocket zostanie zerwane.

  • 4000: Unknown Error - Ogólny błąd. Najczęściej przejściowy.
  • 4001: Unknown Opcode - Błąd w protokole WebSocket.
  • 4003: Not Authenticated - Błędne dane uwierzytelniające.
  • 4004: Authentication Failed - Nieprawidłowy token bota.
  • 4005: Already Authenticated - Próba ponownego uwierzytelnienia, gdy bot jest już połączony.
  • 4007: Invalid Seq - Błąd w sekwencji pakietów podczas wznawiania połączenia.
  • 4008: Rate Limited - Bot wysłał za dużo zapytań WebSocket w krótkim czasie.
  • 4009: Session Timed Out - Bot nie wysyłał "heartbeat" na czas, co spowodowało wygaśnięcie sesji.
  • 4010: Invalid Shard - Niepoprawny numer sharda.
  • 4011: Sharding Required - Bot jest na zbyt wielu serwerach, aby działać bez sharding'u.
  • 4012: Invalid API Version - Bot próbuje użyć nieobsługiwanej wersji API.
  • 4013: Invalid Intents - Próba użycia nieprawidłowych lub wyłączonych intencji.
  • 4014: Disallowed Intents - Próba użycia intencji, które nie są zatwierdzone dla bota.

Błędy programistyczne

Te błędy wynikają z twojego własnego kodu lub używanej biblioteki, a nie z serwerów Discorda.

  • AttributeError: Próbujesz odwołać się do atrybutu, który nie istnieje. Np. ctx.author.id.kick().
  • TypeError: Próbujesz użyć funkcji z nieprawidłowym typem danych, np. dodanie tekstu do liczby.
  • ValueError: Próba użycia wartości, która jest nieprawidłowa, np. podanie ujemnej liczby do funkcji, która wymaga dodatniej.
  • CommandNotFound: Użytkownik wpisał komendę, której nie ma w kodzie.
  • MissingRequiredArgument: Użytkownik nie podał wszystkich wymaganych argumentów dla komendy.
  • CommandInvokeError: Ogólny błąd, który występuje w trakcie wykonania komendy. Szczegóły błędu są w środku i trzeba przejrzeć cały traceback.
  • Client is not ready: Próba wykonania akcji (np. wysłania wiadomości) zanim bot w pełni połączy się z Discordem.
  • Econnreset, EPIPE, ETIMEDOUT: Błędy sieciowe, które mogą występować, jeśli bot ma problemy z połączeniem z serwerami Discorda.
